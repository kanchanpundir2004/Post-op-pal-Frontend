# PostOpPal - Post-Operative Recovery Management System

A modern, responsive healthcare web application for managing post-operative patient recovery with AI-powered assistance.

## Features

### Admin Features

- Dashboard with key metrics and analytics
- Patient management and monitoring
- Staff and user management
- Hospital settings and configuration
- Care plan management and PDF uploads

### Doctor/Nurse Features

- Patient recovery monitoring
- Daily check-in tracking
- Medication compliance monitoring
- Red-flag alerts and notifications
- QR code patient identification

### Patient Features (Web/Mobile)

- Recovery progress tracking
- Daily checklist and reminders
- Symptom reporting
- Care plan access
- QR code for quick identification

## Tech Stack

- **Frontend Framework:** React 18 with TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS + shadcn/ui components
- **Routing:** React Router DOM
- **HTTP Client:** Axios
- **Form Handling:** React Hook Form + Zod
- **State Management:** React Context + React Query
- **Notifications:** React Hot Toast
- **QR Code Scanning:** HTML5 QR Code Scanner
- **Icons:** Lucide React

## Prerequisites

- Node.js 18+ and npm/yarn/pnpm
- Backend API (FastAPI) running on port 8000
